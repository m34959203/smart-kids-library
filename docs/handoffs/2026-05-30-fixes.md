`src/app/api/chatbot/route.ts`: для режимов `general` и `search` перед
`dispatchChat` выполняется FTS по реальной таблице `books` (инлайн через
уже импортированный `getMany`):
```sql
to_tsvector('simple', title_ru||' '||title_kk||' '||title||' '||author||' '||description)
```

названию). Топ-8 подмешиваются в system-prompt + директива: называть ТОЛЬКО
найденные книги, иначе «не нашёл, попробуйте /catalog». В ответ добавлено
поле `groundedCount`. Режим `education` не трогался (школьная программа).
